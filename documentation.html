üñ±Ô∏è Understanding View vs Edit Mode

The Fundamental Concept

View Mode (üëÅÔ∏è Eye Icon) and Edit Mode are the two main states of the app that control how you interact with your canvas:

View Mode üëÅÔ∏è

  • Purpose: Navigation and inspection only
  • Canvas Interaction:
    • Zoom: Pinch to zoom in/out or use scroll wheel
    • Pan: Drag to move around the canvas
    • No Editing: Cannot draw, erase, or modify pixels
  • When to Use:
    • Reviewing your work from different angles
    • Showing your artwork to others
    • Taking screenshots without accidental edits
    • Navigating large canvases

Edit Mode ‚úèÔ∏è

  • Purpose: Active creation and modification
  • Canvas Interaction:
    • All Tools: Full access to drawing, selection, and editing tools
    • Limited Navigation: Basic pan and zoom available
    • Pixel Precision: Click or drag to place pixels
  • When to Use:
    • All drawing and editing activities
    • Creating new artwork
    • Making modifications to existing work

üí° Pro Tip: Switch to View Mode when you want to examine your work without accidentally making changes, then toggle back to Edit Mode when you're ready to continue creating!

üí° Pro Tips & Best Practices

Workflow Optimization

  5. 5. Master Keyboard Shortcuts for faster workflow

Animation Tips

  1. 1. Use Onion Skinning for smooth motion between frames
  2. 2. Start with Keyframes then add in-between frames

  4. 4. Use the Timeline to organize complex animations

  1. 1. Build a Palette before starting major work
  2. 2. Use Gradients for professional shading effects
  3. 3. Experiment with Blend Modes for interesting layer effects
  4. 4. Save Custom Gradients for consistent styling

Export Recommendations

  • Animation: GIF for web, MP4 for video platforms
  • Game Assets: Sprite sheets with proper spacing
